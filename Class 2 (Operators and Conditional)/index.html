1. Difference between Dynamic and Static websites.

Answer :

  1. Static website's data are fixed; they can't be edited by users nor interact with users. Dynamic website's data can be edited by users and they can interact with users.
  2. Static website's are faster than dynamic website's because they don't have to process data and generate pages on the fly. Dynamic website's are slower than static website's because they have to process data and generate pages on the fly.
  3. For example, dynamic websites are Facebook, Twitter, YouTube, etc. users can comment, like, post, share, etc. Static websites include Wikipedia, BBC News, etc.; users can't interact here.
  4. Static website's are cheaper and easier to develop and maintain than dynamic website's because they don't require many resources and they don't have to be updated frequently. Dynamic website's are more expensive and harder to develop and maintain than static website's because they require more resources and frequent updates.
